Graph Display

Description

Program displays 5 sorting algorithm; Bubble Sort, Heap Sort, Insertion Sort, Merge Sort, and Quick sort. All sorts will display a single bar graph representing an array of numbers while Merge Sort will display a second graph representing left and right side arrays being merged and Heap Sort will also display a second display showing the heap binary tree.

Controls

Clicking on the algorithm drop down will open it and let you select which algorithm to run. Adjusting the speed slider changes how fast the algorithm runs in milliseconds (100ms - 1000ms). The size slider will adjust how many elements are sorted in the display (10-200). *NOTE* the size slider will change to 31 if above 31 and heap sort is selected due to display limitations. The Start Sort button runs the algorithm and the reset button resets the display.
